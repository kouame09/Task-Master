# documentation

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) version 16 or higher

### Installing

```bash
npm install
```

### Running

```bash
npm run dev
```

### Building

```bash
npm run build
```

### Testing

```bash
npm run test
```
